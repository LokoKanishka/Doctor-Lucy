# Inventario de Aplicaciones - Doctor Lucy

Este documento contiene la lista completa de aplicaciones detectadas en el sistema, organizadas por su función.

## 🛠️ Utilidades y Sistema
- **Htop**: Monitor de procesos y recursos.
- **Terminal de Ubuntu**: Consola de comandos.
- **Monitor del Sistema**: Versión gráfica para ver recursos.
- **Discos / Creador de Discos**: Gestión de almacenamiento.
- **Configuración**: Panel de control de Ubuntu.
- **Relojes / Calendario / Calculadora**: Herramientas básicas.

## 🤖 Inteligencia Artificial y Desarrollo
- **Visual Studio Code**: Editor de código principal.
- **Antigravity**: Entorno de agentes IA.
- **Ollama / Lucy**: Lanzador local de modelos de lenguaje.
- **NotebookLM**: Herramienta de investigación.
- **Lucy Convertir a TXT**: Script personalizado para procesamiento de archivos.

## 🎬 Multimedia y Gráficos
- **OBS Studio**: Grabación y transmisión de pantalla.
- **Audacious / Rhythmbox**: Reproductores de música libres.
- **Shotwell**: Organizador de fotos.
- **VLC Media Player**: Reproductor de video universal.
- **Spotify**: Streaming de música.
- **Sunshine**: Servidor para streaming de juegos/escritorio.

## 🌐 Navegación y Comunicación
- **Google Chrome**: (Con múltiples perfiles: Amarillo, Naranja, Roja, etc.).
- **Firefox**: Navegador web secundario.
- **Thunderbird**: Cliente de correo electrónico.
- **Remmina**: Cliente de escritorio remoto.

## 📄 Oficina y Documentos
- **LibreOffice (Writer, Calc, Impress)**: Suite de oficina (Word, Excel, PowerPoint).
- **Evince (Visor de Documentos)**: Para PDFs.
- **Editor de Texto (Gedit/gnome-text-editor)**.
